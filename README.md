# Modern TypeScript

*Developer notes about modern (2020) TypeScript.*

#### Rules

- Avoid `any`, use `unknown` instead.
- Avoid `Object` and `object`, use `{[key: string] : unknown}` instead.
- Avoid Enums, use Unions instead.
